## Installation Instructions
1. Install:
   ```bash
   npm install
   ```
2. Create .env file(look .env.example)

3. Edit config/config.yml from ur database

4. Migrate:
    ```bash
    npm run migrate:dev
   ```
5. Seed database:

  ```bash
   npm run seed
  ```

6. Start:
   ```bash
   npm run start:dev
   ```
## Other
1. Swagger url 'http://localhost:8000/api/v1/swagger'
2. Base url 'http://localhost:8000/api/v1'
